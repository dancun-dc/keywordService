# keyword

#### 介绍
keywords项目的后台服务，采用Springboot框架

#### 软件架构
采用springboot+mysql+mybatisplus


#### 安装教程

1.  从版本控制导入项目
2.  配置maven依赖
3.  下载注入依赖
4.  导入数据库

#### 使用说明

1.  maven
2.  install
3.  copy jar包到服务器
4.  配置服务器jdk环境变量 
5.  终端或cmd窗口运行 java -jar XXX.jar或双击jar包(看不到调试信息)
